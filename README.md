# 🚀 JavaScript para QA

Repositório com meus estudos de **JavaScript aplicado à Qualidade de Software (QA)**, com foco em lógica de programação, manipulação de dados e preparação para testes automatizados.

---

## 📚 Conteúdos estudados

- Variáveis (`var`, `let`, `const`)
- Tipos de dados (string, number, boolean, array, object, function)
- Arrays (manipulação e acesso)
- Estruturas de decisão (`if`, `else`)
- Estruturas de repetição (`for`, `forEach`)

---

## 💻 Exercícios

### ✔️ Exercício 1 – Estrutura de repetição

Percorrer uma lista de nomes e exibir:

- Número da execução (iniciando em 1)
- Nome do elemento atual
- Separador para organização da saída

📌 Conceitos aplicados:
- Arrays
- Loop (`for` e `forEach`)
- Controle de índice (`index + 1`)

---

### ✔️ Exercício 2 – Estrutura de decisão

Percorrer uma lista de cidades e verificar se um valor específico está presente (**Florianópolis**).

📌 Conceitos aplicados:
- Condicional (`if/else`)
- Comparação de valores
- Validação de dados dentro de loops
